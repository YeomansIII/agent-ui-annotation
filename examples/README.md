# Examples

Runnable examples for each framework adapter.

## Setup

First, build the main package:

```bash
cd ..
pnpm install
pnpm build
```

Then, in any example directory:

```bash
pnpm install
pnpm dev
```

## Vanilla JS

```bash
cd vanilla
pnpm install
pnpm dev
```

## React

